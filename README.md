
# Assessment Project

A small Python project for assessment tasks. This repository contains a minimal FastAPI-based app and supporting modules used for the assessment exercises.

**Contents**
- **src/**: Python package containing the application modules (`api.py`, `core.py`, `data.py`, etc.).
- **main.py**: Entrypoint for running the application.
- **Answers.md**: Assessment answers and notes.

**Setup**
1. Create and activate the virtual environment (already included under `ass/` if provided):

	python -m venv .venv
	source .venv/bin/activate

2. Install dependencies (adjust for your environment):

	pip install -r requirements.txt

**Run**
- To run the app locally (example):

  ```uvicorn src.api:app --reload```
